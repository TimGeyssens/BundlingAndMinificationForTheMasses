Optimus

Optimus is a UI layer on top of the Asp.net Web Optimization framework for Umbraco, making it possible to create script and style bundles from within the backoffice

Why bundles?

  • Improves request load time
    • Reducing the number of requests
    • Reducing the size of requested assets

How?

  • Asp.net Web Optimization framework
    • Create bundles from codebehind

Optimus

  • Allows you to create bundles from Umbraco UI
  • Both v6 and v7 (maybe v4)

Optimus, more than meets the eye

  • Provider model for dynamic css/js languages
  • Less
  • Sass
  • Coffeescript
  • Typescript
